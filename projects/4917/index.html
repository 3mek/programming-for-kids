4917: Machine Code For Kids - The Card Game

Richard Buckland's 4 bit computer

This is a card game for teaching kids how to read and write machine code, using a simple 4 bit computer with very simple instruction set, 2 general purpose registers and only 16 nibbles of addressable memory.

START THE DEBUGGER

The cards have 54 small programs, illustrating various low level concepts such as:
  1. fetch execute cycle
  2. instruction arguments
  3. variables
  4. pointers
  5. self modifying programs
  6. self corrupting programs
  7. self referencing code
  8. integer overflows and underflows
  9. control flow and addressable memory
  10. code is data
  11. data ia code
  12. stepping through code
